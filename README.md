# 🚀 MeetSync – Real-Time Video Calling Platform

MeetSync is a modern full-stack video conferencing platform built to deliver smooth real-time communication with high-quality video and audio streaming using WebRTC and Socket.IO.

---

✨ Features

✅ Real-Time Video Calling  
✅ High Quality Audio & Video Streaming  
✅ Create & Join Meeting Rooms  
✅ Unique Room ID Generation  
✅ Mute / Unmute Controls  
✅ Camera On / Off Functionality  
✅ Responsive & Modern UI  
✅ Peer-to-Peer Communication  
✅ Real-Time Socket Communication  
✅ Fast & Interactive Experience  
✅ Screen Sharing
✅ Live Chat
✅ Authentication System

---

🛠️ Tech Stack

🎨 Frontend

⚡ Getting Started

🔽 Clone Repository

```bash

```

---

🔧 Backend Setup

```bash

npm install
npm start
```

---

💻 Frontend Setup

```bash
cd frontend/my-react-app
npm install
npm run dev
```

---

🌐 How MeetSync Works

1️⃣ User creates or joins a meeting room  
2️⃣ Socket.IO establishes real-time communication  
3️⃣ WebRTC creates peer-to-peer connection  
4️⃣ Users connect through live audio & video streams
